# Stylelint-Plugin-Color

Stylelint 插件，用于指定项目的颜色格式，**WIP**。

- [x] 支持 `hex`、`hexa`、`rgb`、`rgba` 格式校验
- [ ] 支持 `stylelint --fix` 自动修复

## 配置说明

- `format`: 颜色格式，为以下之一：
  - `hex`: 十六进制格式
  - `hexa`: 十六进制格式（带透明度）
  - `rgb`: RGB 格式
  - `rgba`: RGBA 格式
